Клонирование проекта в папку nameProject:
git clone https://... nameProject

добавление файлов в коммит:
git add .

запись коммита:
git commit -m "комментарий коммита"

Отправить коммит:
git push

принять изменения:
git pull

Авторство коммитов
git config --global user.email "ваш емайл"
git config --global user.name "ваше Имя"
